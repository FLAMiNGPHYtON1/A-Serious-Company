There are two components of this system

-ExpressJS BackEnd
-Vue FrontEnd


To Run this project follow thse steps:


## Requirements
- Node.js (v12 or higher)
- MySQL (v5.7 or higher)
- npm (Node Package Manager)


### Backend Setup
1. Navigate to the `BackEnd` directory in your terminal.
2. Ensure that you have installed all required dependencies. Run the following command:

   	npm install

3. MySQL database should automatically be created, this system runs locally on your system
4. start back-end with this command

	node index.js

### FrontEnd Setup
1. Navigate to the `FrontEnd` directory in your terminal.
2. Ensure that you have installed all required dependencies. Run the following command:

   	npm install

3.Start the frontend development server:

	npm run serve


Basic CRUD principles have been followed

1. Files can be uploaded
2. Files can be deleted
3. Files can be edited
4. Files cna be read (downloaded)
